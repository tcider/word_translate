Task:
Parse word document on Russian (partially translated to English) and choose part, needed to translate.
Post this part to other document for translate.
After translation, make initial document with this translation.

Requirements:
1. Docx documents about 200 pages
2. Documents consists tables and paragraphs, which needed to translate with different format.
3. OS Windows, GUI
4. Batch processing documents (more than 100 docs)

Stack:
Python, docx, docxtpl.
Used some algorithm tricks to fasten processing.
